Provides user-friendly interfaces and interactive workflows for cluster operations including creation, deletion, and cleanup with confirmation prompts and detailed status messages.

## Key Components

- **OperationsUI**: Main struct providing cluster operation interfaces
- **SelectClusterForOperation**: Generic cluster selection with argument parsing or interactive selection
- **SelectClusterForDelete**: Cluster selection for deletion with safety confirmation prompts
- **SelectClusterForCleanup**: Cluster selection for cleanup operations with force mode support
- **ShowOperationStart/Success/Error**: Comprehensive status messaging with troubleshooting tips
- **ShowConfigurationSummary**: Display cluster configuration details before creation

## Usage Example

```go
// Create operations UI service
opsUI := NewOperationsUI()

// Select cluster for generic operation
clusterName, err := opsUI.SelectClusterForOperation(clusters, args, "status")
if err != nil {
    return fmt.Errorf("cluster selection failed: %w", err)
}

// Select cluster for deletion with confirmation
clusterName, err = opsUI.SelectClusterForDelete(clusters, args, false)
if err != nil {
    return err
}

// Show operation progress
opsUI.ShowOperationStart("delete", clusterName)
// ... perform operation ...
opsUI.ShowOperationSuccess("delete", clusterName)

// Display configuration summary
opsUI.ShowConfigurationSummary(config, false, false)
```

The service handles both command-line argument parsing and interactive selection, provides safety confirmations for destructive operations, and offers detailed feedback with troubleshooting guidance when operations fail.
