Provides cross-platform Docker installation and management functionality with special support for Windows WSL2 environments. Handles Docker detection, automatic installation via platform-specific package managers, and Windows-specific WSL2 Docker setup.

## Key Components

- **DockerInstaller**: Main struct providing Docker installation methods across platforms
- **IsDockerRunning()**: Checks if Docker daemon is accessible with Windows WSL2 support
- **IsDockerInstalledButNotRunning()**: Detects Docker installation without running daemon
- **Platform-specific installers**: Dedicated methods for macOS (Homebrew), Linux distributions (apt/yum/dnf/pacman), and Windows (WSL2)
- **Windows WSL2 integration**: Specialized functions for WSL2 Docker setup and command wrapping

## Usage Example

```go
// Create installer instance
installer := docker.NewDockerInstaller()

// Check Docker status
if !installer.IsInstalled() {
    fmt.Println("Docker not found")
    fmt.Println(installer.GetInstallHelp())
    
    // Automatic installation
    if err := installer.Install(); err != nil {
        log.Fatalf("Installation failed: %v", err)
    }
} else if docker.IsDockerInstalledButNotRunning() {
    fmt.Println("Docker installed but not running")
}

// Verify Docker is running
if docker.IsDockerRunning() {
    fmt.Println("Docker is ready")
}
```

The installer automatically detects the platform and uses appropriate installation methods - Homebrew for macOS, native package managers for Linux distributions, and WSL2 Docker Engine setup for Windows environments.
