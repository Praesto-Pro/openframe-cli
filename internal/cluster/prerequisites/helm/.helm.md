A cross-platform Helm installer that automatically detects, installs, and configures Helm across macOS, Linux, and Windows with special WSL2 support for Windows environments.

## Key Components

- **HelmInstaller**: Main installer struct providing platform-agnostic Helm management
- **isHelmInstalled()**: Detects Helm installation with timeout protection and WSL2 support
- **Install()**: Platform-specific installation methods (Homebrew, install script, WSL2)
- **createHelmWrapper()**: Windows-specific wrapper for seamless WSL2 integration
- **commandExists()**: Utility for checking command availability
- **helmInstallHelp()**: Platform-specific installation guidance

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := helm.NewHelmInstaller()
    
    // Check if Helm is already installed
    if !installer.IsInstalled() {
        fmt.Println("Helm not found, installing...")
        
        // Get platform-specific help
        fmt.Println(installer.GetInstallHelp())
        
        // Attempt automatic installation
        if err := installer.Install(); err != nil {
            log.Fatalf("Failed to install Helm: %v", err)
        }
        
        fmt.Println("Helm installed successfully!")
    } else {
        fmt.Println("Helm is already installed")
    }
}
```

The installer handles platform differences automatically - using Homebrew on macOS, the official install script on Linux, and creating a WSL2 wrapper with PATH management on Windows. Special attention is given to Windows environments where Helm runs inside WSL2 but needs to be accessible from Windows command line with proper path conversion and environment variable setup for CI compatibility.
