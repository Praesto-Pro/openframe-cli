Provides cross-platform k3d installation functionality with automatic detection, platform-specific installation methods, and Windows WSL2 support.

## Key Components

- **K3dInstaller**: Main installer struct implementing the installer interface
- **IsInstalled()**: Checks if k3d is already installed on the system
- **Install()**: Platform-aware installation with fallback mechanisms
- **GetInstallHelp()**: Provides platform-specific installation instructions

## Platform Support

- **macOS**: Uses Homebrew (`brew install k3d`)
- **Linux**: Supports multiple package managers (apt, yum, dnf, pacman) with fallback to install script
- **Windows**: Installs k3d in WSL2 Ubuntu and creates a Windows batch wrapper

## Installation Methods

The installer implements multiple fallback strategies:

1. **Package managers**: Platform-native package managers when available
2. **Install script**: Official k3d install script with retry logic
3. **Direct binary**: Downloads k3d binary directly from GitHub releases
4. **AUR packages**: For Arch Linux using yay or paru

## Usage Example

```go
installer := k3d.NewK3dInstaller()

// Check if k3d is already installed
if !installer.IsInstalled() {
    fmt.Println("k3d not found, installing...")
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        // Fall back to manual instructions
        fmt.Println("Automatic install failed:", err)
        fmt.Println(installer.GetInstallHelp())
    }
} else {
    fmt.Println("k3d is already installed")
}
```

## Error Handling

The installer includes robust error handling with:
- Connection timeouts to prevent hanging
- Retry logic for network failures
- Fallback version when GitHub API is unavailable
- Platform detection with unsupported OS handling
