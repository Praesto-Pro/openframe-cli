A cross-platform kubectl installer that handles installation and detection of the Kubernetes CLI tool on macOS, Linux, and Windows systems, with special WSL2 support for Windows environments.

## Key Components

- **KubectlInstaller**: Main struct providing kubectl installation capabilities
- **IsInstalled()**: Checks if kubectl is available on the system with timeout protection
- **Install()**: Platform-specific automatic installation (Homebrew on macOS, binary download on Linux, WSL2 on Windows)
- **GetInstallHelp()**: Returns platform-specific manual installation instructions
- **commandExists()**: Utility function to check if a command is available in PATH
- **createKubectlWrapper()**: Creates Windows batch wrapper for WSL2 kubectl integration

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := kubectl.NewKubectlInstaller()
    
    // Check if kubectl is already installed
    if installer.IsInstalled() {
        fmt.Println("kubectl is already installed")
        return
    }
    
    // Attempt automatic installation
    if err := installer.Install(); err != nil {
        // Fall back to manual instructions
        fmt.Printf("Automatic installation failed: %v\n", err)
        fmt.Printf("Manual installation help:\n%s\n", installer.GetInstallHelp())
        return
    }
    
    fmt.Println("kubectl installed successfully!")
}
```

The installer handles platform-specific quirks like using Homebrew on macOS, direct binary downloads on Linux, and creating WSL2 wrappers with Windows path conversion on Windows systems.
