A K3D cluster manager that provides comprehensive cluster lifecycle management with platform-specific optimizations for Windows/WSL environments. The manager handles cluster creation, deletion, listing, and configuration with automatic kubeconfig management and Docker fallback mechanisms.

## Key Components

- **K3dManager**: Main struct implementing cluster management operations
- **ClusterManager**: Interface defining cluster management methods
- **NewK3dManager**: Constructor for creating manager instances with default timeout
- **NewK3dManagerWithTimeout**: Constructor with custom timeout configuration
- **CreateCluster**: Creates K3D clusters with automatic configuration and validation
- **DeleteCluster**: Removes clusters with Docker fallback for failed operations
- **GetRestConfig**: Returns Kubernetes REST configuration for existing clusters
- **ListClusters**: Retrieves information about available K3D clusters

## Usage Example

```go
import (
    "context"
    "github.com/flamingo-stack/openframe-cli/internal/cluster/models"
    "github.com/flamingo-stack/openframe-cli/internal/shared/executor"
)

// Create a new K3D manager
executor := executor.NewRealCommandExecutor()
manager := NewK3dManager(executor, true) // verbose mode enabled

// Create a cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
    K3s: models.K3sConfig{
        Image: "rancher/k3s:v1.31.5-k3s1",
        Servers: 1,
        Agents: 2,
    },
}

restConfig, err := manager.CreateCluster(context.Background(), config)
if err != nil {
    log.Fatal(err)
}

// List all clusters
clusters, err := manager.ListClusters(context.Background())
if err != nil {
    log.Fatal(err)
}

// Delete a cluster
err = manager.DeleteCluster(context.Background(), "my-cluster", models.ClusterTypeK3d, false)
```

The manager includes platform-specific features like WSL IP detection, inotify limit configuration, kubeconfig permission fixes, and Docker container cleanup for robust cross-platform operation.
