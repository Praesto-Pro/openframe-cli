Provides cluster lifecycle management operations for Kubernetes clusters, primarily K3d, with support for creation, deletion, status checking, and comprehensive cleanup of resources.

## Key Components

### ClusterService
Main service struct that orchestrates cluster operations:
- `manager` - K3d cluster manager for underlying operations
- `executor` - Command executor for system operations
- `suppressUI` - Flag to disable interactive UI elements for automation

### Constructor Functions
- `NewClusterService()` - Creates service with default configuration
- `NewClusterServiceSuppressed()` - Creates service with UI suppression for automation
- `NewClusterServiceWithOptions()` - Creates service with custom K3d manager

### Core Methods
- `CreateCluster()` - Creates new cluster or handles existing cluster gracefully
- `DeleteCluster()` - Deletes cluster with progress feedback
- `ListClusters()` - Returns list of all available clusters
- `GetClusterStatus()` - Retrieves status information for specific cluster
- `GetRestConfig()` - Returns Kubernetes REST configuration for cluster access
- `CleanupCluster()` - Performs comprehensive resource cleanup before deletion

### Cleanup Operations
- `cleanupHelmReleases()` - Removes all Helm releases from cluster
- `cleanupKubernetesResources()` - Deletes resources from common namespaces
- `cleanupDockerResources()` - Cleans up Docker containers and images

## Usage Example

```go
// Create cluster service
executor := executor.NewRealCommandExecutor()
service := cluster.NewClusterService(executor)

// Create a new cluster
config := models.ClusterConfig{
    Name: "my-cluster",
    Type: models.ClusterTypeK3d,
}

restConfig, err := service.CreateCluster(config)
if err != nil {
    log.Fatal(err)
}

// Get cluster status
status, err := service.GetClusterStatus("my-cluster")
if err != nil {
    log.Fatal(err)
}

// Cleanup and delete cluster
err = service.CleanupCluster("my-cluster", models.ClusterTypeK3d, true, false)
if err != nil {
    log.Fatal(err)
}

err = service.DeleteCluster("my-cluster", models.ClusterTypeK3d, false)
if err != nil {
    log.Fatal(err)
}
```

The service handles UI interactions gracefully, providing rich feedback in interactive mode while supporting automation through UI suppression.
