Provides bootstrap functionality for OpenFrame clusters, orchestrating the complete setup process from cluster creation to chart installation.

## Key Components

**Service**: Main bootstrap service that coordinates cluster creation and chart installation
- `Execute()` - Command entry point with flag parsing and validation
- `bootstrap()` - Core orchestration logic for cluster setup
- `createClusterSuppressed()` - Creates K3d cluster with suppressed UI
- `installChartWithMode()` - Installs OpenFrame charts with deployment configuration
- `initializeWSL()` - Windows-specific WSL2 initialization (Ubuntu setup)
- `buildClusterConfig()` - Constructs cluster configuration with defaults

**Deployment Modes**: Supports three deployment configurations:
- `oss-tenant` - Open source tenant deployment
- `saas-tenant` - SaaS tenant deployment  
- `saas-shared` - SaaS shared deployment

**Platform Support**: Cross-platform with Windows WSL2 initialization and Unix-like system support.

## Usage Example

```go
// Initialize bootstrap service
service := NewService()

// Execute bootstrap with flags (typically called by cobra.Command)
cmd := &cobra.Command{
    Use: "bootstrap [cluster-name]",
    Run: func(cmd *cobra.Command, args []string) {
        err := service.Execute(cmd, args)
        if err != nil {
            log.Fatal(err)
        }
    },
}

// Direct bootstrap execution
err := service.bootstrap("my-cluster", "oss-tenant", false, true)
if err != nil {
    log.Fatal(err)
}
```

The service handles the complete OpenFrame setup workflow: WSL2 initialization (Windows), cluster creation with K3d, and chart installation from the default OSS tenant repository.
