This service manages ArgoCD installation and operations within the OpenFrame CLI, providing a high-level interface for ArgoCD lifecycle management through Helm and direct ArgoCD operations.

## Key Components

- **ArgoCD struct**: Main service containing Helm manager, path resolver, ArgoCD manager, and command executor
- **NewArgoCD()**: Constructor that creates a new ArgoCD service with non-verbose internal executor
- **Install()**: Installs ArgoCD using Helm with progress indication and error handling
- **WaitForApplications()**: Waits for all ArgoCD applications to reach ready state
- **IsInstalled()**: Checks if ArgoCD is currently installed in the cluster
- **GetStatus()**: Retrieves current ArgoCD installation status and information

## Usage Example

```go
// Create ArgoCD service
helmManager := helm.NewHelmManager(executor, pathResolver)
pathResolver := config.NewPathResolver()
executor := executor.NewRealCommandExecutor(true, false)

argoCDService := NewArgoCD(helmManager, pathResolver, executor)

// Install ArgoCD
cfg := config.ChartInstallConfig{
    ClusterName: "my-cluster",
    Verbose: true,
}

ctx := context.Background()
if err := argoCDService.Install(ctx, cfg); err != nil {
    log.Fatal(err)
}

// Wait for applications to be ready
if err := argoCDService.WaitForApplications(ctx, cfg); err != nil {
    log.Fatal(err)
}

// Check installation status
installed, err := argoCDService.IsInstalled(ctx)
if err != nil {
    log.Fatal(err)
}
```
