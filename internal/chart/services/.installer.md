Orchestrates the complete Helm chart installation process for OpenFrame, managing ArgoCD and app-of-apps deployments with proper error handling and context support.

## Key Components

- **`Installer`** - Main orchestrator struct that coordinates chart installations
- **`InstallCharts()`** - Primary installation method using default background context
- **`InstallChartsWithContext()`** - Context-aware installation method with cancellation support
- **Dependencies**: `ArgoCDService` and `AppOfAppsService` for specialized installations

## Usage Example

```go
// Create installer with required services
installer := &Installer{
    argoCDService:    argoCDSvc,
    appOfAppsService: appOfAppsSvc,
}

// Install charts with configuration
config := config.ChartInstallConfig{
    ClusterName: "production",
    // ... other config options
}

// Simple installation
if err := installer.InstallCharts(config); err != nil {
    log.Fatalf("Installation failed: %v", err)
}

// With context for cancellation
ctx, cancel := context.WithTimeout(context.Background(), 10*time.Minute)
defer cancel()

if err := installer.InstallChartsWithContext(ctx, config); err != nil {
    log.Fatalf("Installation failed: %v", err)
}
```

The installer follows a sequential approach: installs ArgoCD first, then optionally deploys app-of-apps from a GitHub repository, and finally waits for all applications to reach ready state. Error handling distinguishes between recoverable installation errors and non-recoverable waiting errors.
