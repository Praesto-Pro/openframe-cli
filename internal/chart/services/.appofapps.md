Manages app-of-apps pattern installation from GitHub repositories by orchestrating git cloning and Helm deployment operations.

## Key Components

- **AppOfApps** - Main service struct containing Helm manager, Git repository handler, and path resolver dependencies
- **NewAppOfApps()** - Constructor function that initializes the service with required dependencies
- **Install()** - Core method that clones a chart repository and installs using Helm
- **IsInstalled()** - Checks if the app-of-apps chart is currently installed
- **GetStatus()** - Retrieves current status information for the app-of-apps deployment

## Usage Example

```go
// Initialize dependencies
helmManager := helm.NewHelmManager()
gitRepo := git.NewRepository()
pathResolver := config.NewPathResolver()

// Create app-of-apps service
appOfApps := NewAppOfApps(helmManager, gitRepo, pathResolver)

// Configure installation
config := config.ChartInstallConfig{
    ClusterName: "production",
    AppOfApps: &models.AppOfAppsConfig{
        GitHubRepo:   "https://github.com/company/charts",
        GitHubBranch: "main",
        ChartPath:    "app-of-apps",
    },
}

// Install app-of-apps
err := appOfApps.Install(ctx, config)
if err != nil {
    // Handle installation error
}

// Check installation status
installed, err := appOfApps.IsInstalled(ctx, "default")
```

The service handles temporary directory cleanup, branch validation, and provides detailed error context including cluster information for troubleshooting.
