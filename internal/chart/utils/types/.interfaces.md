This file defines the core interface contracts for the OpenFrame CLI chart installation and management system, establishing a clean separation of concerns across service layers.

## Key Components

### Core Service Interfaces
- **ChartInstaller** - Main orchestrator for complete chart installation workflows
- **ServiceOrchestrator** - Coordinates service interactions during installations
- **WorkflowExecutor** - Executes multi-step workflows with tracking and error handling

### Provider Interfaces
- **ClusterLister** - Abstracts cluster discovery and listing operations
- **HelmProvider** - Manages Helm chart operations including ArgoCD and app-of-apps installations
- **GitProvider** - Handles Git repository cloning and cleanup operations

### Service Component Interfaces
- **ArgoCDService** - Dedicated ArgoCD lifecycle management (install, status, wait operations)
- **AppOfAppsService** - Manages app-of-apps pattern installations and monitoring

### Configuration & UI Interfaces
- **ConfigBuilder** - Constructs installation configurations from parameters
- **PathResolver** - Resolves certificate and configuration file paths
- **OperationsUI** - Provides user interaction capabilities for cluster selection and confirmations
- **ServiceFactory** - Creates service instances with proper dependency injection

## Usage Example

```go
// Example service orchestration
type installer struct {
    orchestrator ServiceOrchestrator
    factory      ServiceFactory
}

func (i *installer) InstallCharts(config config.ChartInstallConfig) error {
    req := InstallationRequest{
        Args:         []string{config.ClusterName},
        Force:        config.Force,
        DryRun:       config.DryRun,
        Verbose:      config.Verbose,
        GitHubRepo:   config.GitHubRepo,
        GitHubBranch: config.GitHubBranch,
    }
    
    return i.orchestrator.ExecuteInstallation(req)
}

// Example workflow execution with step tracking
func (w *workflowExecutor) Execute() *WorkflowResult {
    result := &WorkflowResult{Steps: make([]StepResult, 0)}
    
    w.AddStep("validate-cluster", "Validate cluster connectivity", 
        func() error { return validateCluster() }, true)
    w.AddStep("install-argocd", "Install ArgoCD", 
        func() error { return installArgoCD() }, true)
        
    return result
}
```
