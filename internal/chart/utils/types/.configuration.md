Configuration types and structures for OpenFrame deployment settings including Docker registry, deployment modes, and ingress configurations.

## Key Components

**Core Types:**
- `DeploymentMode` - Enum for deployment types (OSS, SaaS, SaaS Shared)
- `IngressType` - Enum for ingress configurations (localhost, ngrok, GCP)
- `DockerRegistryConfig` - Docker registry authentication settings
- `NgrokConfig` - Ngrok tunnel configuration with IP allowlisting and registration tracking
- `SaaSConfig` - SaaS-specific repository and branch settings
- `IngressConfig` - Combined ingress configuration options
- `ChartConfiguration` - Complete Helm chart installation configuration

**Constants:**
- Deployment mode constants: `DeploymentModeOSS`, `DeploymentModeSaaS`, `DeploymentModeSaaSShared`
- Ingress type constants: `IngressTypeLocalhost`, `IngressTypeNgrok`, `IngressTypeGCP`
- `NgrokRegistrationURLs` - Static URLs for Ngrok dashboard and documentation

**Functions:**
- `GetRepositoryURL()` - Returns appropriate GitHub repository URL based on deployment mode

## Usage Example

```go
// Create deployment configuration
config := &ChartConfiguration{
    DeploymentMode: &DeploymentModeOSS,
    DockerRegistry: &DockerRegistryConfig{
        Username: "myuser",
        Password: "mypass",
        Email:    "user@example.com",
    },
    IngressConfig: &IngressConfig{
        Type: IngressTypeNgrok,
        NgrokConfig: &NgrokConfig{
            AuthToken: "your-auth-token",
            Domain:    "myapp.ngrok.io",
            UseAllowedIPs: true,
            AllowedIPs: []string{"192.168.1.0/24"},
        },
    },
}

// Get repository URL for deployment mode
repoURL := GetRepositoryURL(DeploymentModeOSS)
// Returns: "https://github.com/flamingo-stack/openframe-oss-tenant"
```
