Handles construction and configuration of OpenFrame installation parameters, including Helm values processing and deployment mode branch selection.

## Key Components

- **Builder**: Main configuration builder that integrates with config service and UI operations
- **HelmValues**: Struct representing the YAML structure of Helm values files for OSS and SaaS deployments
- **NewBuilder()**: Factory function that creates a new Builder instance
- **getBranchForDeploymentMode()**: Determines the appropriate Git branch based on deployment mode (OSS, SaaS Tenant, SaaS Shared)
- **BuildInstallConfig()**: Constructs standard installation configuration with app-of-apps setup
- **BuildInstallConfigWithCustomHelmPath()**: Enhanced builder for custom Helm values files and deployment modes

## Usage Example

```go
// Initialize builder with UI operations
operationsUI := chartUI.NewOperationsUI()
builder := config.NewBuilder(operationsUI)

// Build standard install configuration
config, err := builder.BuildInstallConfig(
    false, // force
    false, // dryRun
    true,  // verbose
    "my-cluster",
    "https://github.com/flamingo-stack/openframe-charts",
    "main",
    "/path/to/certs",
)

// Build with custom Helm values and deployment mode
customConfig, err := builder.BuildInstallConfigWithCustomHelmPath(
    false, false, true, false, // force, dryRun, verbose, nonInteractive
    "my-cluster",
    "https://github.com/flamingo-stack/openframe-charts",
    "main",
    "/path/to/certs",
    "/custom/values.yaml",
    "saas-shared", // deployment mode
)
```

The builder automatically reads Helm values files to determine the correct Git branch based on deployment configuration (OSS vs SaaS) and handles certificate directory resolution through the config service.
