Contains configuration structures for Helm chart installation operations in the OpenFrame CLI.

## Key Components

- **ChartInstallConfig**: Main configuration struct that holds all parameters needed for chart installation, including cluster targeting, execution modes, and app-of-apps pattern support
- **HasAppOfApps()**: Method that determines if app-of-apps configuration is properly configured for GitOps workflows

## Usage Example

```go
package main

import (
    "github.com/flamingo-stack/openframe-cli/internal/config"
    "github.com/flamingo-stack/openframe-cli/internal/chart/models"
)

func installChart() {
    // Basic chart installation configuration
    config := &config.ChartInstallConfig{
        ClusterName:    "production",
        Force:          false,
        DryRun:         true,
        Verbose:        true,
        Silent:         false,
        NonInteractive: false,
        SkipCRDs:       false,
    }

    // Configure app-of-apps pattern for GitOps
    config.AppOfApps = &models.AppOfAppsConfig{
        GitHubRepo: "my-org/argocd-apps",
        // Additional app-of-apps configuration...
    }

    // Check if app-of-apps is configured
    if config.HasAppOfApps() {
        // Proceed with GitOps-based installation
        fmt.Println("Installing with app-of-apps pattern")
    }
}
```

The configuration supports various installation modes including dry-run, verbose output, silent operation, and non-interactive execution for CI/CD environments. It integrates with ArgoCD's app-of-apps pattern for GitOps workflows.
