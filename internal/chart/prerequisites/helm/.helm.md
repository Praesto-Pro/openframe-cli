Cross-platform Helm package manager installer with automatic OS-specific installation and Windows WSL2 integration. Provides installation detection, platform-appropriate installation methods, and creates Windows wrappers for WSL2 environments.

## Key Components

- **HelmInstaller**: Main installer struct with methods for installation detection and OS-specific installation
- **isHelmInstalled()**: Detects existing Helm installation with Windows WSL2 support
- **helmInstallHelp()**: Returns platform-specific installation instructions
- **Install()**: Orchestrates OS-specific installation methods
- **Platform installers**: Separate methods for macOS (Homebrew), Linux (package managers), and Windows (WSL2)
- **Windows wrapper creation**: Creates batch file wrapper and PATH management for WSL2 integration

## Usage Example

```go
package main

import (
    "fmt"
    "log"
)

func main() {
    installer := helm.NewHelmInstaller()
    
    // Check if Helm is already installed
    if installer.IsInstalled() {
        fmt.Println("Helm is already installed")
        return
    }
    
    // Show manual installation help
    fmt.Println(installer.GetInstallHelp())
    
    // Or attempt automatic installation
    if err := installer.Install(); err != nil {
        log.Printf("Automatic installation failed: %v", err)
        fmt.Println("Please install manually:", installer.GetInstallHelp())
    } else {
        fmt.Println("Helm installed successfully!")
    }
}
```

The installer handles platform detection, package manager selection (apt, yum, dnf, pacman), and creates WSL2 integration with proper path conversion and environment variable setup for Windows users.
