Cross-platform command executor with specialized WSL support for Windows environments, providing command abstraction, error handling, and recovery mechanisms for WSL and Docker operations.

## Key Components

**Core Interfaces:**
- `CommandExecutor` - Abstraction interface for command execution with dependency injection support
- `ExecuteOptions` - Configuration struct for fine-grained command execution control
- `CommandResult` - Structured result containing exit code, output streams, and execution duration

**WSL Management:**
- `IsWSLAvailable()` / `IsWSLUbuntuAvailable()` - Cached availability checks with thread-safe mutex protection
- `WakeUpWSL()` / `TryRecoverWSL()` - WSL health monitoring and recovery operations
- `RestartDockerInWSL()` - Docker daemon management within WSL2 Ubuntu environments

**Error Handling:**
- `WSLError` - Specialized error type with operation context, exit codes, and actionable suggestions
- WSL-specific exit code constants and error detection patterns
- Platform-aware error suggestions and recovery guidance

**Implementation:**
- `RealCommandExecutor` - Production command executor with dry-run support and Windows/WSL command wrapping
- Thread-safe caching for expensive WSL availability checks
- Timeout handling and context cancellation support

## Usage Example

```go
// Create executor with dry-run and verbose options
executor := NewRealCommandExecutor(false, true)

// Basic command execution
result, err := executor.Execute(ctx, "kubectl", "get", "pods")
if err != nil {
    if wslErr, ok := err.(*WSLError); ok {
        fmt.Printf("WSL Error: %s\n", wslErr.Suggestion)
    }
    return err
}

// Advanced execution with custom options
result, err = executor.ExecuteWithOptions(ctx, ExecuteOptions{
    Command: "helm",
    Args:    []string{"install", "app", "./chart"},
    Dir:     "/workspace",
    Env:     map[string]string{"KUBECONFIG": "/home/user/.kube/config"},
    Timeout: 5 * time.Minute,
})

// WSL health management
if IsWSLAvailable() && !IsWSLUbuntuAvailable() {
    if err := TryRecoverWSL(); err != nil {
        log.Printf("WSL recovery failed: %v", err)
    }
}
```
