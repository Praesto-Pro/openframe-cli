Provides utilities for managing and discovering Kubernetes namespaces, including resource discovery across namespaces to locate services.

## Key Components

- **GetNamespaces()** - Retrieves all available namespaces in the current cluster
- **ValidateNamespace()** - Verifies if a specified namespace exists
- **FindResourceNamespace()** - Intelligently discovers the namespace containing a service by searching deployments, statefulsets, and pods
- **searchResourceByType()** - Internal helper that searches for specific resource types across all namespaces with name matching logic

## Usage Example

```go
// Get all namespaces
namespaces, err := provider.GetNamespaces(ctx)
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Available namespaces: %v\n", namespaces)

// Validate a specific namespace exists
err = provider.ValidateNamespace(ctx, "production")
if err != nil {
    log.Printf("Namespace validation failed: %v", err)
}

// Find which namespace contains a service
namespace, err := provider.FindResourceNamespace(ctx, "my-service")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("Service found in namespace: %s\n", namespace)
```

The `FindResourceNamespace` function is particularly useful for service discovery, automatically searching deployments, statefulsets, and pods with intelligent name matching (exact match, prefix with hyphen, or simple prefix). It defaults to the "default" namespace if no matching resources are found.
