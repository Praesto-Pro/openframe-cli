This Go file provides a UI component for discovering and selecting Skaffold configuration files within a project structure.

## Key Components

- **`SkaffoldUI`** - Main UI handler for Skaffold configuration discovery and selection
- **`ServiceSelection`** - Represents a selected service with its name, file path, and directory
- **`SkaffoldCategory`** - Organizes Skaffold files by type (OpenFrame Services, Integrated Tools, Client Apps, etc.)
- **`SkaffoldFile`** - Represents individual skaffold.yaml configuration files

**Main Methods:**
- `DiscoverAndSelectService()` - Discovers available Skaffold files and prompts user selection
- `findSkaffoldYamlFiles()` - Recursively searches for skaffold.yaml/skaffold.yml files
- `categorizeSkaffoldFiles()` - Groups files by project structure (services, tools, client apps)
- `extractServiceName()` - Extracts clean service names from file paths

## Usage Example

```go
// Create a new SkaffoldUI instance
skaffoldUI := NewSkaffoldUI(true) // verbose mode enabled

// Discover and select a service
selection, err := skaffoldUI.DiscoverAndSelectService()
if err != nil {
    if err == ErrNoSkaffoldFiles {
        log.Println("No skaffold.yaml files found in project")
        return
    }
    log.Fatalf("Service selection failed: %v", err)
}

// Use the selected service
fmt.Printf("Selected service: %s\n", selection.ServiceName)
fmt.Printf("Service directory: %s\n", selection.Directory)
fmt.Printf("Skaffold file: %s\n", selection.FilePath)
```

The component categorizes services based on OpenFrame's project structure and provides an interactive selection interface with search filtering capabilities.
